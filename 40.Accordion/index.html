JavaScript
What is JavaScript?
JavaScript, often abbreviated as JS, is a programming language that is one of the core technologies of the World Wide Web, alongside HTML and CSS. Over 97% of the websites use JavaScript on the client side for web page behaviour, often incoporating third-party libraries.
Java and JavaScript
Key differences between Java and JavaScript: Java is an OOP programming language while Java Script is an OOP scripting language. Java creates applications taht run in a virtual machine or browser while JavaScript code is run on a browser only. Java code needs to be compiled while JavaScript code are all in text.
Can I learn JavaScript without Java?
You do not need to know Java to learn JavaScript. Despite the names, the two languages have very little in common and are used for completely different areas of development. You can learn either language first but laarning Java is not a prerequisite to picking up JavaScript. libraries.
JavaScript and ECMAScript
JavaScript is a general purpose scripting language that conforms to the ECMAScript specification. The ECMAScript specification is a blueprint for creating a scripting language. JavaScript is an implementation of that blueprint. On T=the whole, JavaScript implements the ECMAScript specification as described in ECMA-262.
